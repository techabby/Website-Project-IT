ABOUT

This website is designed as a simple and user-friendly restaurant site that showcases desi food items such as biryani, karahi, BBQ, and more. It includes a clean homepage, a menu table page, a reservation form, and an about section. The website provides easy navigation, basic styling, and a clear layout so visitors can explore the restaurant’s items, learn about it, and submit reservations. All pages are created using HTML, CSS and JavaScript with a consistent theme and design.
